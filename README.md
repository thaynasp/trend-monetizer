# Trend Monetizer MVP

Este projeto é um sistema autônomo de detecção de tendências e monetização por afiliados.

## Nicho: Saúde na Menopausa (2026)
O site é gerado automaticamente a partir de arquivos Markdown na pasta `site/content/`.

### Como usar:
1. Adicione um novo arquivo `.md` em `site/content/`.
2. Faça o push para o GitHub.
3. O GitHub Actions irá gerar o HTML em `docs/` e atualizar o site live.

Site Live: [https://thaynasp.github.io/trend-monetizer/](https://thaynasp.github.io/trend-monetizer/)
